About this site

How it's hosted, how traffic reaches it, and future plans.

Hosting
Self-hosted homelab
This site runs on my own server. The host is Ubuntu Server running under Proxmox, deployed using Docker Compose.
Routing
Cloudflare Tunnel > HAProxy
Public traffic is proxied through Cloudflare and enters via a Cloudflare Tunnel. Requests flow: Cloudflare > tunnel > HAProxy > containers. No inbound ports are exposed to the internet.
TLS
Terminated at Cloudflare
HTTPS is handled at the Cloudflare edge. Internally, traffic is routed over my private, internal, networks.
Deployments
Automated Git updates
The repo on GitHub is the source of truth. The server runs an automated git pull on a fixed interval (every 5 minutes) done as a bash script run using a systemd timer.
