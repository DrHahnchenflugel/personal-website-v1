What this is & why I built it

This is both a portfolio site showcasing my projects and a technical demo of my web development and hosting capabilities. The main focus is on the projects themselves, but I also want to share how the site is built and hosted for anyone interested in the technical details. The site is designed to be minimal and modern, with a focus on performance and accessibility.

Hosting

Hardware: Custom-built server
Platform:Proxmox VE
Host OS: Ubuntu Server
Deploy: Docker Compose

Routing

  • Flow: Web > Cloudflare > CF Tunnel > HAProxy > Docker-Compose Container service the site.
  • Inbound Ports: None exposed to the internet
  • TLS, HTTPS: TLS is terminated at Cloudflare. HTTPS is handled at the Cloudflare edge. Internally, traffic stays on private networks.

Deployments

  • Source of truth: GitHub
  • Update method: systemd timer running a bash script that pulls from Github every 5 minutes.
