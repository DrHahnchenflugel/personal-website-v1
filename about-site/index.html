About this site

How it’s hosted, how traffic reaches it, and what’s next.

Hosting

Self-hosted homelab

Platform: Proxmox
Host OS: Ubuntu Server
Deploy: Docker Compose

Routing

Cloudflare Tunnel → HAProxy

Public traffic is proxied through Cloudflare and enters via a Cloudflare Tunnel.

Flow: Cloudflare → tunnel → HAProxy → containers
Inbound ports: None exposed to the internet

TLS

Terminated at Cloudflare

HTTPS is handled at the Cloudflare edge. Internally, traffic stays on private networks.

Automated Git updates
