Automated Windows Deployment Scripts

A set of powershell and bash scripts that help in the setup of a computer from OOBE. Implemented as part of Automated Windows Deployment Process.

< Back to Projects GitHub

What it is

As part of my efforts to fully automate the deployment of Windows, I realised there were some steps that could not be done as part of the OOBE setup (as far as I could see), and had to be done after the fact. Partially, this was because some of these steps required computer restarts between them - if i try to rename my PC and join the domain right after, weird stuff would occur. Here's what I needed done manually:

  • STAGE 1:
  • Task: Rename the computer as dictated by it's asset tag.
  • Restart.
  • STAGE 2:
  • Task: Join the domain with specific settings.
    • Domain Host: Specific server FQDN, based on location.
    • Domain Host: Specific Organisational Unit (OU), based on computer type.
    • Specific admin username/password.
  • Task: Set specific firewall rules to allow PDQ Deploy/Inventory to work.
    • Enable File and Printer Sharing.
    • Enable ICMPv4 Inbound Ping.
  • Restart.
  • STAGE 3:
  • Task: Update Group Policy.
  • Task: Delete all the files once finished.

So, 3 stages, and I have to restart between each one. I create a batch script for each stage. Then, as part of the batch script for Stage 1, I schedule a task that is going to execute Stage 2 to run after the restart (ONLOGON). Ditto for Stage 2 into Stage 3.
A few of the tasks are easier to do with powershell, as opposed to group policies. Namely, in Stage 2, the domain needs to be joined using Add-Computer, and the firewall rules need updating.

Here's the structure: 

Stage1_Rename.bat
|
|_Stage2_JoinDomain.bat
    |_JoinDomain.ps1
    |_FwallRules.ps1
    |
    |_Stage3_GpUpdate.bat

The Code

Stage1_Rename.bat


          
powershell -ExecutionPolicy Bypass -Command "$P = Read-Host 'The PC Name is on its back. Please enter it here:';Rename-Computer -NewName $P -Force"
schtasks /Create /TN "Stage2_JoinDomain" /TR "C:\Scripts\Stage2_JoinDomain.bat" /SC ONLOGON /RL HIGHEST /F
shutdown /r /t 5


Stage2_JoinDomain.bat


          
del /f /q "C:\Scripts\Stage1_Rename.bat"
powershell -ExecutionPolicy Bypass -File "C:\Scripts\JoinDomain.ps1"
del /f /q "C:\Scripts\JoinDomain.ps1"
powershell -ExecutionPolicy Bypass -File "C:\Scripts\FwallRules.ps1"
del /f /q "C:\Scripts\FwallRules.ps1"
schtasks /Create /TN "Stage3_GpUpdate" /TR "C:\Scripts\Stage3_GpUpdate.bat" /SC ONLOGON /RL HIGHEST /F
schtasks /Delete /TN "Stage2_JoinDomain" /F
shutdown /r /t 5

FwallRules.ps1


          
Write-Host "Enabling File Sharing..."

Enable-NetFirewallRule -DisplayGroup "File and Printer Sharing"

Write-Host "Enabled!"
Write-Host "Enabling ICMP4 Ping..."

try{
    Set-NetFirewallRule -Name "FPS-ICMP4-ERQ-In" -Enabled True
    Enable-NetFirewallRule -Name "FPS-ICMP4-ERQ-In"
    Write-Host "Enabled!"
} catch {
    Write-Host "Failed to find rule with name FPS-ICMP4-ERQ-In. Creating alternative..."
    New-NetFirewallRule -Name "Allow ICMPv4 Inbound"`
        -DisplayName "Allow ICMPv4 Inbound Echo Request"`
        -Protocol ICMPv4 -IcmpType 8`
        -Direction Inbound`
        -Action Allow`
        -Profile Any
    Write-Host "Created alternative!"
}

Write-Host "Done!"


joinDomain.ps1


          
while ($true){
  # Prompt for computer type
  Write-Host "Select computer type:"
  Write-Host "1 - REDACTED1"
  Write-Host "2 - REDACTED2"
  Write-Host "3 - REDACTED3"
  Write-Host "4 - REDACTED4"
  $selection = Read-Host "Enter number (1-4)"
  $ou = '-1'
  # Determine OUPath
  switch ($selection) {
      "1" { $ou = "";
          break }
      "2" { $ou = "" ;
          break }
      "3" { $ou = "" ;
          break }
      "4" { $ou = "" ;
          break }
      default {
          Write-Host "Invalid selection."
      }
  }
  if ($ou -ne '-1'){
      break
  }
}

while ($true){
    # Prompt for location
    Write-Host "Select location:"
    Write-Host "1 - REDACTED1"
    Write-Host "2 - REDACTED2"
    Write-Host "3 - REDACTED3"
    Write-Host "4 - REDACTED4"
    $loc = Read-Host "Enter number (1-4)"
    $server = '-1'
    # Determine OUPath
    switch ($loc) {
        "1" { $server = "";
            break }
        "2" { $server = "" ;
            break }
        "3" { $server = "" ;
            break }
        "4" { $server = "" ;
            break }
        default {
            Write-Host "Invalid selection."
        }
    }
    if ($server -ne '-1'){
        break
    }
}

# Prompt for instructor password
$plainPassword = Read-Host "Enter  password" -AsSecureString
$username = "]"

# Build credential
$cred = New-Object System.Management.Automation.PSCredential ($username, $plainPassword)

# Join the domain with specified OU
Add-Computer -ComputerName $env:COMPUTERNAME `
            -DomainName "" `
            -OUPath $ou `
            -Credential $cred `
            -Server $server `
            -ErrorAction Stop

Write-Host "Computer has been added to the domain in OU: $ou"
